Note: In a grid layout, content must be placed inside the colums(.col and .col-*) and only columns may be immediate children of rows(.row). Rows should be placed inside containers for proper padding and alignment


Tip: Grid column widths are set in percentages, so they are always fluid and sized relative to their parent element. In adding, each column has horizontal padding(called gutter) for controlling space between individual columns
